① 課題名
仕事中の親と留守番中の子どもをつなぐ
“音声 × テキスト” ハイブリッドチャットアプリ

② 課題内容（どんな作品か）
「音声で伝えたい子ども」 と「文字で受け取りたい親」
という、コミュニケーション手段の異なる2者をつなぐための双方向チャットアプリ です。

● アプリが解決したい課題
シングルマザーの友人は、仕事中に5歳と9歳の子どもから頻繁に電話があり、
着信だけでは緊急性の判断ができないため、商談中でも席を外して折り返す状況に困っている様子です。
一方で、未就学児〜低学年の子どもは、大人のように入力をスムーズに行うことが難しく、文字でのコミュニケーションに負担があります。
そこで、親の仕事中の精神的・時間的コストを軽減する、子どもの寂しさや不安を少しでも和らげる、
双方の心理的負担を減らすことを目的に設計しました。

● 機能と仕様
【子どもモード】
メッセージは音声録音で送信
親から届いたメッセージは自動で音声読み上げ
シンプルで直感的に操作できるUI

【親モード】
メッセージはテキストで送信
子どもからの音声はテキストに変換されて表示

⑤工夫した点・こだわった点
■ 子ども側：音声中心のシンプル体験
「押して話すだけ」で送信できる、誤操作しないようにシンプルなUIを意識
親のテキストは自動で読み上げ、子どもの読む負担をゼロにした

■ 親側：短時間で内容を理解できる設計
仕事中で音声を聞く余裕がない親のため、子どもの音声は再生するのではなくテキスト化して表示
